How to rerun the build if no combinations to rerun. This configuration takes effect only when you check "Rerun build only for failed parts on the matrix".
Rerun the matrix with all combinations
Reruns whole the matrix. This is useful if you want always rerun something.
Rerun the matrix with no combinations
Reruns the matrix with no children. This is useful if you want rerun only aggregation processes. Some processes may fail if no children.
Don't rerun
Doesn't rerun the matrix. This is useful especially when used with the regular expressions.
